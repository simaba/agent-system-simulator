# Contributing

Thanks for your interest in improving this repository.

## Scope

This repository is a lightweight runnable simulator for multi-agent control logic. Helpful contributions improve clarity, correctness, and usability.

## Good contributions

Examples include:
- improving agent logic,
- clarifying simulator behavior,
- strengthening evaluation output,
- adding well-scoped scenarios,
- correcting inaccuracies.

## Contribution expectations

Please keep contributions:
- concise,
- implementation-aware,
- easy to test,
- free of proprietary or confidential material.
